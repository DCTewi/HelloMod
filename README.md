# MyDevMod

It's a Mod for Minecraft 1.12.2 with Forge.  
It's DCTewi's Minecraft Mod Development Attempt.  

##What's here:  

**Blocks:**  

- An Ore  
- A Block  

**Items:**  

- An Ingot  
- A Sword  
- A Pickaxe  
- An Axe  
- A Shovel

**Others:**  

- A Custom Creative Tab
- English and Chinese Support

*Recipes is same as minecraft tools.

##Building

```.\gradlew build```
